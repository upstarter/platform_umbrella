    Auth.GuardedController

    Use this module in a controller to take the advantage of having the subject of authentication (eg.: an authenticated user) injected in the action as the third argument.

    Usage example

    defmodule RealWorldWeb.MyController do use RealWorldWeb, :controller use RealWorldWeb.GuardedController plug Guardian.Plug.EnsureAuthenticated def index(conn, params, current_user) do

    # ..code..

    end end
